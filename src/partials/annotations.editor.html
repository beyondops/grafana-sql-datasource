Annotation Query Format

Your query should return four standard columns (case sensitive):

    * time: Time since UNIX Epoch in milliseconds. (required)
    * title: The title for the annotation tooltip. (required)
    * text: Text for the annotation. (optional)
    * tags: Tags for the annotation. (optional)

MySQL:
SELECT
  ROUND(UNIX_TIMESTAMP(time_column) * 1000) AS time,
  text_column AS text,
  tag_column AS tags,
  title_column AS title
FROM yourtable
ORDER BY time ASC

PostgreSQL:
SELECT
ROUND(extract(EPOCH FROM time_column) * 1000) AS time,
  text_column AS text,
  tag_column AS tags,
  title_column AS title
FROM yourtable
ORDER BY time ASC
